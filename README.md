# How to use

Compile with maven ``mvn clean package`` or download from <> and execute the jar  
``java -jar data-compressor.jar <encode | decode> <input file> <output file>``

# Algorithm explanation

RLE <-> BWT <-> MTF <-> RLE <-> A0
